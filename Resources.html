The Heart|Resoures

The Resoures we used to model the Heart:

  1. To make the respositories we used a program called Git
  2. To collaborate we used a program called Github
  3. To make the heart and artery we used Blender
  4. To make the heart be a pure functioning heart we used Unity
  5. To make the webpage pretty and have enough text we used Komodo
  6. To send our code off to the web we used a program called SourceTree
  7. To film the movie we used a Pansonic Camera
  8. To edit and make the movie we used Imovie
